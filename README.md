# Kaeng master (Here Pup Pha Kaeng)

โปรเจกต์นี้คือระบบเกมไพ่แคงในรูปแบบ Web-based Multiplayer Card Game ที่รองรับผู้เล่นสูงสุด 5 คนต่อห้อง โดยทำงานบนสถาปัตยกรรมแบบ Client-Server ซึ่งขับเคลื่อนด้วยภาษา C++ ในส่วนของ Backend และใช้ HTML/JavaScript ในการแสดงผลฝั่ง Frontend

## การตั้งค่าและการรันระบบ

**1. การคอมไพล์ซอร์สโค้ด**
เปิด Command Prompt หรือ Terminal ในโฟลเดอร์โปรเจกต์ และใช้คำสั่งต่อไปนี้เพื่อคอมไพล์โค้ด:

```bash
g++ main.cpp -o main -lws2_32

```

**2. การเปิดเซิร์ฟเวอร์ (Running the Server)**
รันไฟล์ที่ได้จากการคอมไพล์เพื่อเริ่มต้นระบบ:

```bash
.\main.exe

```

*(เซิร์ฟเวอร์จะเปิดรับการเชื่อมต่อที่พอร์ต `18080`)*

**3. การเข้าใช้งาน (Accessing the Game)**

* **สำหรับเล่นในเครื่องเดียวกัน (Localhost):** เปิดเบราว์เซอร์และไปที่ `http://localhost:18080`
* **สำหรับเล่นผ่านเครือข่ายจำลอง (เช่น Radmin VPN / LAN):** ให้ผู้เล่นคนอื่นเข้าผ่าน IP Address ของเครื่องโฮสต์ ตามด้วยพอร์ต (ตัวอย่าง: `http://26.x.x.x:18080`)

## กติกาพื้นฐาน (Game Rules)

วัตถุประสงค์หลักของเกมคือ ผู้เล่นต้องบริหารจัดการให้ไพ่ในมือของตนเองมีแต้มรวมน้อยที่สุดในวง

**การนับแต้มไพ่**

* **ไพ่ 10, J, Q, K** มีค่าเท่ากับ 10 แต้ม
* **ไพ่ A** มีค่าเท่ากับ 1 แต้ม
* **ไพ่เลข 2-9** มีแต้มตามหน้าเลขของไพ่

**กติกาและการดำเนินเกม**

* **การเริ่มเกม:** ผู้เล่นคนแรกที่เข้าห้องจะได้รับสิทธิ์เป็น "หัวหน้าห้อง" ซึ่งสามารถตั้งค่าเงินเดิมพันและเลือกผู้แจกไพ่ (Dealer) ได้ เมื่อเริ่มเกม ระบบจะหักเงินเดิมพันเข้ากองกลาง และแจกไพ่ให้ผู้เล่นคนละ 5 ใบ
* **การจั่วไพ่ (Draw):** หยิบไพ่ 1 ใบจากกองกลาง
* **การทิ้งไพ่ (Discard):** สามารถทิ้งไพ่กี่ใบก็ได้ใน 1 เทิร์น แต่ไพ่ที่ทิ้งพร้อมกันจะต้องมีแต้มเท่ากันทั้งหมด (เช่น ทิ้งไพ่ 3 ตอง หรือ 5 คู่)
* **การไหลไพ่ (Flow):** หากไพ่ที่ผู้เล่นต้องการทิ้ง มีแต้มตรงกับไพ่ใบบนสุดของกองทิ้ง ผู้เล่นจะได้รับสิทธิ์ "ไหล" ซึ่งระบบจะทำการหักเงินค่าปรับจากผู้เล่นที่ทิ้งไพ่ก่อนหน้า มามอบให้ผู้ที่ทำการไหลไพ่ทันที
* **การแคง (Call Kaeng):** หากผู้เล่นประเมินว่าไพ่ในมือมีแต้มน้อยที่สุดในวงแล้ว สามารถกด "แคง" เพื่อขอดูไพ่และจบเกม
* **แคงสำเร็จ:** หากแต้มน้อยที่สุดจริง ผู้แคงจะเป็นผู้ชนะและได้รับเงินกองกลางทั้งหมด
* **แคงล้มเหลว (ถูกแหก):** หากมีผู้เล่นคนอื่นในวงที่มีแต้มน้อยกว่าหรือเท่ากับผู้แคง ผู้แคงจะต้องเสียค่าปรับชดเชยให้แก่ผู้เล่นทุกคนในวง


* **ระบบจับเวลา:** ผู้เล่นมีเวลา 20 วินาทีต่อเทิร์น หากไม่ดำเนินการตามเวลาที่กำหนด ระบบจะหักเงินค่าปรับและบังคับเล่นอัตโนมัติ

  
## ขั้นตอนการเล่นและปุ่มควบคุม (Game Manual & Controls)

### การเตรียมตัวและเริ่มเกม (Game Setup)
* **การเข้าห้อง:** เมื่อเปิดหน้าเว็บ จะพบหน้าต่าง "ไพ่แคง Casino" ให้พิมพ์ชื่อเล่นในช่อง `ตั้งชื่อเล่นของคุณ (EN/TH)` แล้วคลิกปุ่ม **`เข้าห้อง`**
* **การตั้งค่า (สำหรับหัวหน้าห้อง):** คนแรกที่เข้าห้องจะได้สิทธิ์เป็นหัวหน้าห้อง (Host) โดยอัตโนมัติ ซึ่งจะต้องตั้งค่าเริ่มต้น ได้แก่ เงินเริ่ม, เดิมพัน, ค่าไหล, ค่าปรับ และคลิกเลือก **ผู้แจกไพ่ (Dealer)** ที่ต้องการ จากนั้นกดปุ่ม **`บันทึกการตั้งค่า`**
* **การเริ่มเกม:** เมื่อผู้เล่นเข้ามาครบตามที่ต้องการแล้ว หัวหน้าห้องสามารถคลิกปุ่ม **`เริ่มเกม!`** ที่มุมขวาบนของหน้าจอเพื่อแจกไพ่

### การดำเนินเกมในแต่ละเทิร์น (Gameplay & Inputs)
เมื่อถึงเทิร์นของคุณ จะสังเกตเห็น **กรอบสีทอง** รอบพื้นที่เล่นของคุณ และมีตัวจับเวลา 20 วินาทีแสดงขึ้นมา คุณสามารถทำ Action ได้ดังนี้:

* **การจั่วไพ่ (Draw):** คลิกเมาส์ซ้ายที่ **กองจั่ว** (หลังไพ่สีน้ำเงิน) บริเวณกลางหน้าจอ 1 ครั้ง เพื่อหยิบไพ่ขึ้นมือ
* **การเลือกไพ่บนมือ:** คลิกเมาส์ซ้ายที่ไพ่ของคุณบริเวณ "มือของคุณ" ไพ่ที่ถูกเลือกจะเด้งลอยขึ้นมาพร้อมกรอบสีทอง (หากเปลี่ยนใจสามารถคลิกซ้ำเพื่อเอาลงได้)
  * *> หมายเหตุ: หากต้องการทิ้งไพ่หลายใบพร้อมกันเป็นชุด (คู่/ตอง) จะต้องคลิกเลือกไพ่ที่มีหน้าไพ่เหมือนกันเท่านั้น*
* **การทิ้งไพ่ (Discard):** หลังจากคลิกเลือกไพ่ที่ต้องการทิ้งแล้ว ให้คลิกปุ่ม **`ทิ้งไพ่`** (สีเขียว) ที่แผงควบคุมด้านล่าง
* **การแคง (Call Kaeng):** หากประเมินแล้วว่าแต้มไพ่ในมือน้อยที่สุดในวง (โดยที่คุณต้อง *ยังไม่ได้จั่วไพ่* ในเทิร์นนั้น) ให้คลิกปุ่ม **`แคง!`** (สีแดง) เพื่อขอเปิดไพ่และจบเกมทันที

### การจบเกม (Game Over)
* เมื่อมีผู้เล่นกดแคง ระบบจะเปิดหน้าไพ่ของทุกคน (จากที่เคยแสดงสถานะว่า "คว่ำ" จะเปลี่ยนเป็นหน้าไพ่จริง) พร้อมแสดงแต้มรวมเหนือชื่อผู้เล่นแต่ละคน
* ระบบจะคำนวณเงินกองกลางและแจกจ่ายให้ผู้ชนะอัตโนมัติ พร้อมแสดงข้อความแจ้งเตือน (Toast) กลางหน้าจอว่าผู้เล่นคนใด **"แคงสำเร็จ"** หรือ **"โดนแหก"**
* หลังจากจบตา หัวหน้าห้องสามารถกดปุ่ม **`ตั้งค่าเกมหน้า`** หรือ **`เริ่มเกม!`** บริเวณขวาบน เพื่อเริ่มเล่นรอบต่อไปได้ทันที

## สถาปัตยกรรมของระบบ (System Architecture)

ระบบถูกออกแบบโดยแยกการทำงานออกเป็น 2 ส่วนหลัก ได้แก่:

### 1. Backend (C++ API Server)

* **Web Server:** ใช้ไลบรารี `httplib.h` ในการจัดการ HTTP Request
* **State Management:** รวมตรรกะของเกม (Game Logic) ไว้ใน `KaengGame.h` ซึ่งรับหน้าที่จัดการไพ่ การคิดคำนวณเงินกองกลาง และการตรวจจับเวลาเทิร์น
* **OOP Dealer System:** ประยุกต์ใช้แนวคิด Inheritance และ Polymorphism ในการสร้างระบบคนแจกไพ่ (Dealer) ที่มีพฤติกรรมแตกต่างกัน (เช่น สุ่มปกติ, ล็อคแจกไพ่สูง, ล็อคแจกไพ่ต่ำ)
* **Connection Management:** มีระบบ Heartbeat ตรวจสอบสถานะผู้เล่น หากพบว่าผู้เล่นปิดเบราว์เซอร์หรือขาดการเชื่อมต่อเกิน 10 วินาที ระบบจะนำผู้เล่นออกจากห้องและปรับสถานะโต๊ะโดยอัตโนมัติ

### 2. Frontend (HTML / JavaScript)

* **Data Synchronization:** ใช้ JavaScript `fetch()` เพื่อส่งคำขอแบบ Polling ไปยังเซิร์ฟเวอร์ทุกๆ 1 วินาที เพื่อนำข้อมูลล่าสุดมาอัปเดตหน้าจอผู้เล่น (Real-time UI)
* **Security & Anti-Cheat:** ระบบส่งข้อมูลจาก Backend จะทำการกรองข้อมูล (Masking) โดยซ่อนข้อมูลหน้าไพ่ของผู้เล่นคนอื่นเป็นคำว่า "คว่ำ" จนกว่าเกมจะจบ เพื่อป้องกันการทุจริตผ่านการตรวจสอบ Network Traffic ของเบราว์เซอร์
